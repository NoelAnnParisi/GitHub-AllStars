# GitHub-AllStars
Who's GitHub reigns supreme? This single page web app allows you to battle your friends & discover the most popular GitHub repos. 

## Click [here]( https://github-battle-dbc9c.firebaseapp.com/) to check it out! 

## Created with:
* React 
* React Router
* EcmaScript 2015
* Axios
* Webpack


## Deployed on:
* Firebase
